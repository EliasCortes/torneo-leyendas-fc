# 🏆 Torneo Leyendas FC

Aplicación en Python para gestionar un torneo personalizado de leyendas del fútbol inspirado en FIFA/FC, con sistema de ruletas, equipos y simulación.

---

## 🚀 Descripción

Este proyecto permite crear y gestionar un torneo entre jugadores (Elias vs Álvaro) con:

- Equipos personalizados
- Sistema de ruleta (Diamante, Oro, Plata, Bronce)
- Fichajes limitados
- Jugador franquicia
- Simulación de partidos
- Persistencia de datos en JSON

Está diseñado como base para evolucionar a una aplicación web completa.

---

## 🧠 Funcionalidades principales

- 🎯 Generación de equipos
- 🎲 Sistema de ruleta con probabilidades
- 💾 Guardado y carga de torneo (`torneo_leyendas.json`)
- ⚽ Gestión de jugadores por niveles
- 🧩 Lógica modular separada (models, lógica, guardado)
- 🖥️ Ejecución por consola (actualmente)
